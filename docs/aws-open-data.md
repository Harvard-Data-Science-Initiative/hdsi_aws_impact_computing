## Overview

The AWS Open Data Program makes high-value, cloud-optimized datasets publicly available at no cost to end users. AWS partners with universities, government agencies, and research organizations to:

- **Democratize access** — datasets available globally for analysis on AWS
- **Lower barriers** — cloud-native formats and tools reduce cost of working with data at scale
- **Build communities** — shared datasets foster cross-institutional collaboration

All datasets are discoverable on the [Registry of Open Data on AWS](https://registry.opendata.aws) and [AWS Data Exchange](https://aws.amazon.com/data-exchange).

Through the **Open Data Sponsorship Program**, AWS covers storage and data transfer costs for accepted datasets for **2 years** (renewable).

## How to Participate

### Option 1: Self-List (data already on AWS)

1. Fork `github.com/awslabs/open-data-registry`
2. Add a YAML metadata file describing your dataset
3. Submit a pull request — AWS reviews and merges
4. Dataset appears on the Registry within days

### Option 2: Sponsorship Program (Recommended)

1. Apply at **[application.opendata.aws](https://application.opendata.aws)**
2. Describe dataset: size, format, update frequency, audience, and research impact
3. AWS evaluates value, community benefit, and data quality
4. If accepted, AWS optimizes and hosts the data on S3
5. Storage + egress covered for **2 years** (renewable)

---

## Strong Application Criteria

| Criterion | What AWS Looks For |
|---|---|
| **High Value** | Enables significant research or serves broad communities |
| **Cloud-Optimized** | Parquet, Zarr, COG, STAC formats (or willingness to convert) |
| **Clear Licensing** | CC0, CC-BY, or Open Government licenses |
| **Active Maintenance** | Regular updates with a clear steward |
| **Community Impact** | Existing or potential user base (downloads, citations) |
| **Documentation** | Metadata, data dictionaries, usage examples |
| **Scale** | Typically > 1 TB where hosting costs are a real barrier |
